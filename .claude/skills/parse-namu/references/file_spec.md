# NH나무증권 해외주식거래내역 .xls 파일 스펙

## 파일 포맷

- 확장자: `.xls` (실제로는 HTML 테이블)
- 인코딩: EUC-KR (`<meta http-equiv="Content-Type" content="text/html; charset=EUC-KR">`)
- 파일명: `NH나무증권_{계좌번호}_{YYMMDD-YYMMDD}.xls`

## HTML 구조

```
<table>
  <thead> 2행 (헤더) </thead>
  <tbody>
    <tr> 메인행 (15 td) </tr>   ← 거래 1
    <tr> 서브행 (13 td) </tr>   ← 거래 1 상세
    <tr> 메인행 (15 td) </tr>   ← 거래 2
    <tr> 서브행 (13 td) </tr>   ← 거래 2 상세
    ...
  </tbody>
</table>
```

## 거래유형 분류

### 정상 디코딩 (현재 파일 기준)

| 거래유형 | 상세내용 예시 | 처리 |
|----------|-------------|------|
| 매수 | 외화증권매수 | 포함 |
| 매도 | 외화증권매도 | 포함 |
| 입금 | 외화배당금입금, 배당금 | 포함 (유형→배당) |
| 입금 | 예탁금이용료, 이벤트 지원금 | 포함 (유형→입금) |
| 입금 | 이체입금, 대체입금 | **제외** (계좌 간 이체) |
| 출금 | 이체출금, 대체출금 등 | **제외** |
| 환전 | 외화매수 | **제외** |
| 입고 | 공모주입고 | **제외** |
| 매수/매도 | 발행어음*, 코스피*, KOSDAQ* | CMA/국내주식이면 **제외** |

### 깨진 인코딩 fallback

일부 내보내기 시점에서 인코딩이 깨질 수 있음:

| 깨진 패턴 | 실제 유형 |
|-----------|-----------|
| ŵ 또는 \xC5\xB5 | 매도 |
| ż 또는 \xC5\xBC | 매수 |
| ȯ 또는 \xC8\xAF | 환전 |
| Ա 또는 \xD4\xB1 | 입금 |
| 뱄 또는 킹 | 이체 |
