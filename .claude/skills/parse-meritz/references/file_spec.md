# 메리츠증권 종합거래내역 파일 스펙

## 파일 개요

| 항목 | 값 |
|------|-----|
| 형식 | OLE2 Microsoft Excel (.xls) |
| 인코딩 | EUC-KR |
| 시트명 | 거래내역조회 |
| 헤더 행 수 | 2 |
| 열 수 | 15 |

## 헤더 구조 (2행)

| idx | 행 0 (주 헤더) | 행 1 (보조 헤더) |
|-----|--------------|----------------|
| 0 | 거래일자 | 거래적요 |
| 1 | 종목코드 | 종목명 |
| 2 | 수량 | 단가 |
| 3 | 수수료 | 제세금 |
| 4 | 세전이자 | 신용이자 |
| 5 | 거래금액 | 반영금액 |
| 6 | 미수발생/변제금 | 융자/대주금액 |
| 7 | 질권자 | 질권설정수량 |
| 8 | 처리지점 | 거래번호 |
| 9 | 거래원번 | 관련계좌 |
| 10 | 예수금잔고 | 유가증권잔고 |
| 11 | 거래원금 | 원금잔액 |
| 12 | 매수일자(질권설정일/출금고제한일) | 만기일자 |
| 13 | 경과일수 | 적용금리 |
| 14 | 통화구분 | 외화예수금잔고 |

## 데이터 행 구조

데이터 행은 **짝/홀 쌍**으로 거래 1건을 이룬다.

- **짝수 행 (주 데이터)**: 거래일자, 종목코드, 수량, 거래금액, 통화구분
- **홀수 행 (상세 데이터)**: 거래적요, 종목명, 단가, 반영금액

### 파싱 추출 필드

| 필드 | 행 | 열 idx | 비고 |
|------|-----|--------|------|
| 거래일자 | 짝수 | 0 | '2026/01/27' → '2026-01-27' |
| 거래적요 | 홀수 | 0 | 유형 판단 기준 |
| 종목코드 | 짝수 | 1 | 'SGOV.AX' → suffix 제거 → 'SGOV' |
| 종목명 | 홀수 | 1 | 비고(비교) 컬럼에 저장 |
| 수량 | 짝수 | 2 | 문자열이면 쉼표 제거 후 float |
| 단가 | 홀수 | 2 | |
| 거래금액 | 짝수 | 5 | 쉼표 포함 가능: '1,609.75' |
| 통화구분 | 짝수 | 14 | 'USD' 또는 '' |

## 거래 유형별 데이터 특성

| 거래적요 | 거래금액(짝수) | 반영금액(홀수) | 포함 여부 |
|---------|-------------|-------------|---------|
| 해외주식매수대금 | USD 금액 (예: '1,609.75') | 동일 | ✅ → 매수 |
| 해외주식매도대금 | USD 금액 | 동일 | ✅ → 매도 |
| 해외주식매수 | '0' | 0.0 | ❌ (금액 없음) |
| 해외주식매도 | '0' | 0.0 | ❌ (금액 없음) |
| 배당금 | USD 금액 또는 '0' | USD 금액 | ✅ → 배당 |
| 종목교체입고 | '0' | 0.0 | ✅ → 입고 (수량 기록용) |
| 종목교체출고 | '0' | 0.0 | ✅ → 출고 (수량 기록용) |
| 환전외화매수(자체) | USD 금액 | KRW 금액 | ❌ |
| 전자계좌입금 | KRW 금액 (쉼표 포함) | 동일 | ❌ |
| 배당세금출금(원화) | '0' | KRW 금액 | ❌ |
| 오픈뱅킹출금 | KRW 금액 | 동일 | ❌ |

## 종목코드 규칙

메리츠증권은 Reuters/거래소 suffix를 종목코드에 포함한다.

| suffix | 거래소 | 예 |
|--------|--------|---|
| .AX | ASX (호주) | SGOV.AX, IREN.AX |
| .NY | NYSE | SGOV.NY, CRCL.NY, INFQ.NY |
| .OQ | NASDAQ | IREN.OQ |

suffix 제거 패턴: `re.sub(r"\.[A-Z]{2,3}$", "", code)`

## 샘플 데이터

### 해외주식매수대금 (2026-01-27)
```
짝수행: ['2026/01/27', 'SGOV.AX', '16', 0.0, 0.0, '1,609.75', 0.0, '', '디지털센터', '', ...]
홀수행: ['해외주식매수대금', 'ISHARES 0-3M TREASURY BOND', '100.6093', 0.0, 0.0, 1609.75, ...]
```
→ 거래일자=2026-01-27, 유형=매수, 종목코드=SGOV, 수량=16, 단가=100.6093, 금액=1609.75, 통화=USD

### 배당금 (2026-02-05)
```
짝수행: ['2026/02/05', 'SGOV.AX', '0', 0.0, 0.0, '6.49', 0.0, ..., 'USD']
홀수행: ['배당금', 'ISHARES 0-3M TREASURY BOND', '0.0000', 0.0, 0.0, 6.49, ...]
```
→ 거래일자=2026-02-05, 유형=배당, 종목코드=SGOV, 수량=0, 금액=6.49, 통화=USD

### 종목교체 (2026-02-23)
```
짝수행: ['2026/02/23', 'SGOV.AX', '34', ..., '0', ..., 'USD']
홀수행: ['종목교체출고', 'ISHARES 0-3M TREASURY BOND', '100.5214', ...]

짝수행: ['2026/02/23', 'SGOV.NY', '34', ..., '0', ..., 'USD']
홀수행: ['종목교체입고', 'ISHARES 0-3M TREASURY BOND', '100.5214', ...]
```
→ SGOV.AX 34주 출고 + SGOV.NY 34주 입고 (거래소 이전, 순포지션 불변)
